Hva er Bedring?

Bedring er en tjeneste der man kjapt og enkelt skal kunne finne behandlere i sitt nærområde. Appen har en kartløsning der du kan se alle personlige trenere, fysioterapeuter og ernæringsfysiologer innenfor ditt område. Målet med Bedring er å gjøre det enkelt å både finne en behandler, men også å sette opp time. I tillegg skal det være enklere for behandlerne å bli oppdaget. Behandlerne har også full kontroll over sine egne priser, og timeplan.

Teknisk informasjon

API

APIet er bygd med Node som runtime engine, i ExpressJS. Som database har jeg tatt i bruk MongoDB og Mongoose. Registrering- og Innloggingssystemet er lagd med Passport, med JWT for å verifisere requests. Jeg bruker Stripe sin API som betalingsløsning. Appen og websiden deler samme API.

Frontend

Designet er lagd i Figma, med bruk av ulike plugins som Mockup. Websiden er lagd med React/NextJS.
